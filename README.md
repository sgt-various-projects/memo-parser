# memo-parser

## 概要
空行区切りテキスト と Org-Mode 風の アウトライン付きのテキストを正規化して
編集し、再度 空行区切りテキスト と Org-Modeの アウトライン付きのテキストに再構成する。

## 機能詳細

### 入力機能
説明なし

### 展開機能
Org-Mode風のアウトライン `** ` を解釈して展開する。
`** `で始まる行をoutline_lv2とする。

コードフェンス(```から次の```まで)は空行で展開しない。

### 編集機能
タイトル入力欄は他タイトル入力欄の内容からリスト補完する機能を持つ。
（入力文字列を含む候補を表示する。（大文字小文字は無視。入力と完全一致する候補は除外））

タイトル入力欄を矩形ドラッグするとダイアログを表示し、一括入力可能。


#### キーボード・マウス操作

| 操作 | 動作 |
|------|------|
| タイトル の末尾で `→` | 同行の コンテンツ にフォーカス |
| コンテンツ の先頭で `←` | 同行の タイトル の末尾にフォーカス |
| コンテンツ 最終行で `↓` / 先頭行で `↑` | 上下の コンテンツ にフォーカス|
| 補完リスト表示中 `↑` / `↓` | 候補のハイライト移動 |
| 補完リスト表示中 `Enter` | 選択した候補を確定 |
| `Ctrl` + クリック（タイトル） | 複数 タイトル を選択/解除 |
| 空白部分のドラッグ（タイトル） | 矩形内の タイトル を一括選択し、プロンプトで一括入力（キャンセル時は選択解除）|


### エクスポート機能
画面上 と ファイルへのエクスポート機能を持つ。


エクスポート時、
- 同じタイトルの複数行は、1 つの見出し `** タイトル` の下に本文を順に連結する（各本文の後に空行 2 行）
- タイトルが空の行は '未整理' として扱う
- タイトル '未整理' は末尾に出力する。(未実装?)

### 補助機能(未実装部分あり)
　
- ソート機能
- ダークモード切替機能（初期状態はダークモード）
- インポート欄の表示/非表示 機能（初期状態はインポート欄を表示）


### キーボード・マウス操作(全体)

| 操作 | 動作 |
|------|------|
| `Ctrl` + ホイール | ブラウザのズームではなくページスクロール |
| `Ctrl` + `R` | ブラウザ標準のリロードを無効化 |
| 画面上部の `reload` ボタン | 画面をリロード |
